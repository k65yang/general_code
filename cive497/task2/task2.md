## Problem 3 - Issues in Sampling
a) Quantiziation is the process of constraining continous data to a discrete set. For example, continous data can be constrained as intergers, so values such as 4.1 and 4.2 would be rounded down and simply represented as 4. This process will induce accuacy errors as decimal points are not longer being stored. These types of errors occur when analog-digital converters do not have sufficient bins. To avoid this error, if possible, get converters with a greater amount of bins. Else, adjust the range to more appropriate values such that the errors would be minimized.  

b) Clipping errors occur when the amplitude of the true signal exceeds the measurement range of the device. As a result, what is represented in the time domain graphs are horizontals lines near the peaks and troughs of the measurement record. To avoid this error, adjust the range in the measurement instrument to be able to measure all the signal. If that is not possible, get better instruments.  

c) Oversampling is when the sampling frequency is much greater than what is required to reconstruct the true frequency. As an example, if the true frequency was 2 Hz, oversampling would be using a sampling frequency of 200 Hz. It is not necessary to use a sampling frequency that high. Oversampling means more data to be processed, which leads to larger file sizes and longer computation times.

The Matlab code to generate the data for the above examples is pasted below for inspection.  

```matlab
syms t

% Define parameters for 5 hz sine wave
f = 5;
A = 2;
y = A*sin(2*pi*f*t);

% Generate base signal (continous)
ncyle = 1;
Fsa = 1000; % # of samples per a second
ta = 0:1/Fsa:ncyle;
ya = subs(y, t, ta);

% Generate aliasing example
Fsa_alias = 8;
ta_alias = 0:1/Fsa_alias:ncyle;
ya_alias = subs(y, t, ta_alias);

% Generate alising plot
figure;
p_base = plot(ta, ya, 'b-', 'linewidth', 2); hold on;
p_alias = plot(ta_alias, ya_alias, 'ro', 'linewidth', 2);
p_alias_temp = plot(ta_alias, ya_alias, 'r-', 'linewidth', 1);
title('Signal damaged from aliasing'); 
xlabel('Time'); ylabel('Amplitude');
axis([0 ncyle -A-0.1 A+0.1]);
legend([p_base, p_alias], {'True Signal', 'Sampled Signal'});
hold off;

% Generate quantization example
Fsa_quan = 80;
ta_quan = 0:1/Fsa_quan:ncyle;
ya_quan = subs(y, t, ta_quan);

bin_size = 2;
ya_quan = bin_size * round(ya_quan/bin_size);

% Generate quantization plot
figure;
p_base = plot(ta, ya, 'b-', 'linewidth', 2); hold on;
p_quan = plot(ta_quan, ya_quan, 'ro', 'linewidth', 2);
p_quan_temp = plot(ta_quan, ya_quan, 'r-', 'linewidth', 1);
title('Signal damage from quantization'); 
xlabel('Time'); ylabel('Amplitude');
axis([0 ncyle -6 6]);
legend([p_base, p_quan], {'True Signal', 'Sampled Signal'});
hold off;

% Generate clipping example
Fsa_clip = 30;
ta_clip = 0:1/Fsa_clip:ncyle;
ya_clip = subs(y, t, ta_clip);

ya_clip(ya_clip > 1.5) = 1.5;
ya_clip(ya_clip < -1.5) = -1.5;

% Generate clipping plot
figure;
p_base = plot(ta, ya, 'b-', 'linewidth', 2); hold on;
p_clip = plot(ta_clip, ya_clip, 'ro', 'linewidth', 2);
p_clip_temp = plot(ta_clip, ya_clip, 'r-', 'linewidth', 1);
title('Signal damage from clipping'); 
xlabel('Time'); ylabel('Amplitude');
axis([0 ncyle -A-0.1 A+0.1]);
legend([p_base, p_clip], {'True Signal', 'Sampled Signal'});
hold off;

% Generate oversampling example
Fsa_over = 250;
ta_over = 0:1/Fsa_over:ncyle;
ya_over = subs(y, t, ta_over);

% Generate oversampling plot
figure;
p_base = plot(ta, ya, 'b-', 'linewidth', 2); hold on;
p_over = plot(ta_over, ya_over, 'ro', 'linewidth', 2);
title('Signal Oversampling'); 
xlabel('Time'); ylabel('Amplitude');
axis([0 ncyle -A-0.1 A+0.1]);
legend([p_base, p_over], {'True Signal', 'Sampled Signal'});
hold off;
```

syms k integer
